# Stompy Mwendwa - Portfolio Website

Professional portfolio for Senior DevOps & Platform Reliability Engineer.

## Tech Stack
- Next.js 14+
- Tailwind CSS
- React

## Content Structure
All content is stored in `/src/data/` directory:
- `branding.js` - Name, title, tagline, contact info, key metrics
- `about.js` - Biography, certifications, education, languages
- `competencies.js` - 9 core competencies with descriptions
- `caseStudies.js` - 5 detailed case studies
- `services.js` - 3 service offerings

CV data is stored in `/public/cv.json`

## Getting Started
npm install
npm run dev

Open http://localhost:3000

## Pages (Placeholders)
- `/` - Homepage
- `/about` - About page
- `/competencies` - Competencies page
- `/case-studies` - Case studies page
- `/services` - Services page

Note: All pages are currently placeholders. UI design will be implemented separately.
